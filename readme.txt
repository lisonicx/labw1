Программа реализует сортировку массива методами Bubble Sort, Selection Sort и Insertion Sort. Размерность массива вводится пользователем с клавиатуры. Массив заполняется случайными числами в диапазоне от 0 до 999 включительно. В программе используются шаблоны.

Пример работы программы:

Введите размерность массива: 10

Исходный массив: 327 2 395 849 644 633 317 209 543 824
Отсортированный массив методом BubbleSort: 2 209 317 327 395 543 633 644 824 849
Отсортированный массив методом SelectionSort: 2 209 317 327 395 543 633 644 824 849
Отсортированный массив методом InsertionSort: 2 209 317 327 395 543 633 644 824 849
